Open Street Mic

The core tenets of this service are:

  • Provide a simple service that works with any telephone
  • Minimal logging and no monetization of user traffic and activity
  • In country numbers for as many places as we can setup
This project utilizes a collection of open-source technologies, formats and protocols, including:
  • Asterisk PBX
  • IceCast2 Streaming Server
  • SIP and DID Virtual Phone and Number Systems
